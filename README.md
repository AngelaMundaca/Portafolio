## 📌 Descripción

Portfolio web profesional desarrollado para presentar mis proyectos como **Desarrolladora Full Stack JavaScript Trainee**.  

Incluye una sección de presentación personal, tecnologías, y proyectos destacados como:

- 🍽️ Buscador de Recetas (consumo de API)
- 📋 Kanban Pro (aplicación fullstack)

El enfoque del sitio es ser claro, visualmente atractivo y fácil de navegar para reclutadores.
